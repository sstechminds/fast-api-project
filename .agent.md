# Pandas to Polars Migration Guide

Polars is a DataFrame library designed for high performance. This guide helps you transition from Pandas to Polars smoothly.

## Installation
To install Polars, use:

```bash
pip install polars
```

## DataFrame Creation

### Pandas
```python
import pandas as pd

df = pd.DataFrame({
    'a': [1, 2, 3],
    'b': [4, 5, 6]
})
```

### Polars
```python
import polars as pl

df = pl.DataFrame({
    'a': [1, 2, 3],
    'b': [4, 5, 6]
})
```

## Basic Operations
### Selecting Columns

#### Pandas
```python
selected = df[['a']]
```

#### Polars
```python
selected = df.select('a')
```

## Grouping
### Pandas
```python
grouped = df.groupby('a').sum()
```

### Polars
```python
grouped = df.groupby('a').agg(pl.sum('b'))
```

## Conclusion
This is a brief guide for transitioning from Pandas to Polars. The syntax varies but the overall logic remains similar.
